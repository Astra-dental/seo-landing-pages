# Internal Linking Plan

Strategy for connecting location landing pages, service pages, and core site pages
so link equity flows efficiently and users can navigate naturally.

## Link Hierarchy

```
Homepage
  -> /langley-dental-services/        (services hub)
       -> individual service pages
  -> location pages (dentist-{area})
       -> relevant service pages
       -> nearby location pages
  -> /about-the-dentist/
  -> /contact-us/
```

## Rules of Thumb

- Every location page links UP to the services hub and to /contact-us/.
- Each location page links to 2-4 of the most relevant service pages.
- Cross-link nearby locations (e.g. Murrayville <-> Brookswood <-> Langley) to
  build a local cluster, but keep it relevant (same region only).
- Use descriptive anchor text ("cosmetic dentistry in Langley"), not "click here".
- Avoid orphan pages: every published page must be linked from at least one other.
- Keep the footer service-area list as a site-wide hub linking all location pages.

## Anchor Text Bank

- "book a dental appointment in {location}"
- "family dentist serving {location}"
- "cosmetic dentistry near {location}"
- "emergency dentist in {location}"
- "learn about our {service}"
